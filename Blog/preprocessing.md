# Preprocessing   
## Term Freq - Inverse Doc Freq (TF-IDF)   
- Bag of word algo → order of words doesn’t matter
   
- Computer freq of words → TF
   
- If word appears in all docs → IDF = 0
   
- If word appears in most docs → IDF < if word appears in less (in their respective doc
   
- TF-IDF = TF \* IDF
